* Build 새롭게 생성하는 방법

1. 새롭게 생성하고자 하는 이름의 프로젝트 생성 (콘솔 응용 프로그램/ 솔루션용 디렉터리 만들기 해제/ 다음 선택 후 빈프로젝트 생성)
2. (기존 LAFLAS_3DOF 등)원본 폴더와 같은 형태로 구성
3. 프로젝트 소스파일에 해당 파일 추가 (원본 Source 폴더 내 확장자 .c)
4. 프로젝트 헤더파일에 해당 파일 추가 (원본 Source 폴대 내 확장자 .h)
5. SNOPT 라이브러리를 생성한 "프로젝트" 하위에 추가
6. 솔루션 우클릭 -> 구성관리자 -> Release, 새로만들기 x64
7. 프로젝트 우클릭 -> 속성 -> 구성속성 -> 링커 -> 입력 -> 특정 기본라이브러리 무시에
"libcmt.lib" 입력

- VS 2013 상위 버전 사용 시,
프로젝트 우클릭 -> 구성 -> 일반 -> 플랫폼도구집합 v120 선택 -> 솔루션 다시빌드
(v120 Tool이 없는 경우 따로 설치해야함, VS2013 설치하면 Tool이 자동적으로 설치되므로 VS2013 설치를 권장, Tool 만 따로 설치 가능하나 Error 발생하는 경우 자주 있음)
